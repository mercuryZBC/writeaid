# WriteAid-知识记录与分享平台
## 项目简介
前后端分离的个人知识库平台，支持用户创建和发布文档、搜索功能、多级评论、点赞功能。支持用户查看最近浏览、最近编辑、最近评论的文档。
## 使用技术
前端：使用React 框架与 Ant Design UI组件库进行开发
后端：
开发语言：Go
web开发框架：Gin
orm框架：Gorm
数据库：Mysql
缓存：Redis
搜索引擎：ElasticSearch
